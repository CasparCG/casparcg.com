---
title: Scene Producer
---

The scene producer was a means of creating xml based templates in CasparCG 2.1 and was the backend for a .psd producer.

**Note: It is no longer included in releases since 2.2+** (Although it _might_ be added back sometime. See [Server#708](https://github.com/CasparCG/server/issues/708))

For documentation only this XML definition document exists: https://github.com/CasparCG/server/blob/2.1.x/core/producer/scene/scene.xsd
