---
title: HTML Producer
---

The HTML Producer allows you to load a hosted HTML page into CasparCG. Unlike [HTML Templates](https://github.com/CasparCG/help/wiki/Media%3A-HTML-Templates), pages loaded through the HTML Producer can't be controlled by CasparCG.

To use the HTML Producer in a `PLAY` command, type something like this: `PLAY 1-1 [HTML] "https://deniosoftware.com"`

For a more comprehensive guide to building templates in HTML please see [this guide](https://chrisryanouellette.gitbook.io/casparcg-html-template-guide) by CasparCG user Christopher Ouelette.
